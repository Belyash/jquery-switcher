jQuery Switch Plugin

jQuery Switch Plugin

List of presets:

  • useWrap - default true
  • status - default 0
  • leftLabel - default ON
  • rightLabel - default OFF
  • onTurnOn - callback, when status change to 1
  • onTurnOff - callback, when status change to 0

Default Call

            $('#switcher_1').switcher();

With Custom Label

            $('#switcher_2').switcher({
                leftLabel: "Moscow",
                rightLabel: "New York"
            });

Default ON

            $('#switcher_3').switcher({
                status: 1
            });

Callbacks

            $('#switcher_4').switcher({
                onTurnOff: function() {
                    alert('Switcher: Turn OFF');
                },
                onTurnOn: function() {
                    alert('Switcher: Turn ON');
                }
            });
